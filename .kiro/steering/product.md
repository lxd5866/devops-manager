# DevOps Manager 产品概述

DevOps Manager 是一个分布式运维管理系统，专为大规模基础设施管理而设计，支持 10,000+ 客户端节点并发连接。

## 核心能力

- **Agent 管理**: 长连接心跳监控，实时掌握节点状态
- **任务分发**: 批量命令执行、文件分发、脚本部署
- **主机组织**: 节点分组、标签管理、标签化运维
- **Web 界面**: RESTful API 和 Web UI 提供远程操作和监控
- **高可用性**: 基于 MySQL 和 Redis 的可扩展架构

## 系统架构

系统采用客户端-服务端模型：
- **Agent**: 部署在被管理服务器上，负责命令执行、文件传输和状态上报
- **Server**: 中央控制平面，负责 Agent 管理、任务调度和结果汇总
- **Web/API**: 管理界面，提供 Swagger 文档化的操作和监控接口

## 主要特性

- 实时主机状态监控，包含系统指标（CPU、内存、磁盘）
- 批量任务执行，支持超时控制和结果跟踪
- 文件传输功能，具备 MD5 完整性校验
- 基于 WebSocket 的 Web 终端支持
- 完整的 API 文档，集成 Swagger UI
- 多语言支持（中文/英文文档）
